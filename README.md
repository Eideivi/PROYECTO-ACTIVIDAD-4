Descripción del proyecto

Este proyecto consiste en una simulación de un sistema de emergencias médicas que analiza datos de ECG para detectar eventos críticos, notificar al hospital y recalcular la ruta de atención. Además, incorpora conceptos de programación concurrente mediante el uso de asincronía en Python.

Problema abordado

El sistema busca representar cómo una aplicación puede monitorear señales médicas, reaccionar ante eventos graves y mantener un control del estado general del proceso.

Solución implementada

El programa genera datos simulados de ECG, clasifica el evento detectado, envía una notificación al hospital, recalcula la ruta según la gravedad y evalúa el estado del sistema mediante una lógica de semáforo.
